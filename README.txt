Page Replacement Demonstration

Features:
------------------------------------------------------------------------------------------
- This is a simple demo program of Page replacement algorithm that includes three implemented algorithm namely (FIFO,OPT,LRU).s
- Input to the program is input.txt file which includes algorithm name, no.of page frames and Reference string respectively to be used as an input.
- Program implements an algorithm and calculates the number of page faults occured in each type of algorithm as well as  output of page frames available in memory after each page replacement.
